Conversor Online de Vídeos para MP3 e MP4

Este projeto é um site que oferece um conversor online de vídeos para MP3 e MP4. Com um design elegante, a página apresenta um menu horizontal com as opções para conversão de vídeos, bem como uma seção sobre a empresa e um link para contato.
O fundo da página é composto por uma imagem que ocupa toda a área visível, o que traz um aspecto mais profissional para o site. O código HTML também inclui a definição de classes para a formatação das caixas de menu e uma classe para a caixa central que contém as opções de conversão.
O projeto foi desenvolvido em 2023 e traz uma abordagem moderna para um conversor online de vídeo.
